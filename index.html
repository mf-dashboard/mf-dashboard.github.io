How to Get Your CAS File
  1. Visit CAMS Online or use the myCAMS mobile app
  2. Select "Detailed Statement"
  3. Choose "With Zero Balance" or "Without Zero Balance"
  4. Enter your email address and set a password
  5. You'll receive the CAS PDF via email within minutes
  6. Upload it here and enter the same password

Currently supports CAMS CAS files.
